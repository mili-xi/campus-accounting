# 校园场景化记账小程序项目说明

## 项目概述

这是一个专为大学生设计的校园场景化记账小程序，提供预算管理、消费记录、财务分析和风险识别功能。

## 功能特点

1. **预算管理**：设置每月预算，自动按周拆分
2. **记账功能**：记录收入和支出，支持分类管理
3. **财务分析**：提供收支统计和图表分析
4. **风险识别**：识别校园贷风险关键词，提供安全提示
5. **预警系统**：设置消费预警阈值，及时提醒用户

## 技术架构

### 前端技术栈
- 微信小程序原生开发
- WXML + WXSS + JavaScript
- Canvas 图表绘制

### 后端服务
- 微信小程序云开发
- 云数据库（存储记账记录）
- 云存储（存储图片和文件）
- 云函数（处理复杂业务逻辑）

## 真机测试常见问题

### 问题：云开发无法使用
**解决方案**：
1. 检查云开发环境是否已开通
2. 确认云开发环境ID是否正确
3. 在 app.js 中检查云开发初始化代码

### 问题：图片无法加载
**解决方案**：
1. 检查图片链接是否为 HTTPS 协议
2. 确认图片域名已添加到合法域名列表
3. 检查图片是否存在且可访问

### 问题：网络请求失败
**解决方案**：
1. 检查服务器域名是否已配置
2. 确认接口地址是否正确
3. 检查网络连接是否正常

### 问题：页面白屏
**解决方案**：
1. 检查页面路径是否正确
2. 查看控制台错误信息
3. 检查数据初始化是否成功

## 调试方法

### 使用真机调试
1. 在微信开发者工具中点击"真机调试"
2. 使用微信扫码在真机上打开小程序
3. 在工具中查看 console 输出和网络请求

### 数据模拟
项目支持数据模拟功能，在 app.js 中设置 `useCloud: false` 即可启用
```javascript
// app.js 中修改云开发设置
wx.cloud.init({
  env: wx.cloud.DYNAMIC_CURRENT_ENV, // 或手动设置环境ID
  traceUser: true
})
```

## 注意事项

1. **数据安全**：定期备份云数据库数据
2. **性能优化**：避免在页面加载时执行大量计算
3. **用户体验**：保持页面加载时间在 1 秒以内
4. **兼容性**：确保小程序在不同设备上的兼容性

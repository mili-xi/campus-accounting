# 校园场景化记账小程序

一个专为在校大学生设计的校园场景化记账小程序，提供预算管理、消费记录、财务分析和风险识别功能。

## 功能特点

### 1. 记账表单
- 日期选择
- 支出/收入类型选择
- 金额输入
- 预设校园场景化分类
- 备注输入

### 2. 预算管理
- 生活费预算设置
- 自动按周拆分预算
- 预算进度展示
- 剩余预算提醒

### 3. 消费预警
- 单日支出大于300元预警
- 一周超预算预警
- 单笔大额（大于500元）风险提示

### 4. 校园贷风险识别
- 备注关键词过滤（校园贷、分期贷、秒批、无抵押、低息贷款、花呗套现等）
- 危险关键词强制提示
- 固定安全提示页面

### 5. 财务分析
- 总支出/总收入计算
- 支出分类饼图
- 收入分类饼图
- 每日消费趋势折线图
- 结余计算

### 6. 数据管理
- 记录列表展示
- 记录筛选和搜索
- 记录编辑和删除
- 加载更多功能

## 技术实现

### 前端技术
- 微信小程序原生开发
- WXML + WXSS + JavaScript
- 云开发（Cloud Development）
- Canvas图表绘制

### 后端服务
- 微信小程序云开发
- 云数据库（MongoDB）
- 云函数
- 云存储

## 风险识别

### 危险关键词列表

- 分期贷
- 秒批
- 无抵押
- 低息贷款

### 预警规则

## 使用方法

### 1. 开发环境搭建
1. 下载并安装微信开发者工具
2. 克隆或下载项目代码
3. 打开微信开发者工具，导入项目
4. 配置云开发环境

### 2. 云开发初始化
1. 在微信开发者工具中点击"云开发"
2. 创建云开发环境
3. 复制环境ID到app.js中
4. 创建数据库集合

### 3. 运行项目
1. 在微信开发者工具中点击"编译"
2. 在模拟器中测试功能
3. 使用真机调试功能测试

## 部署发布

1. 在微信开发者工具中点击"上传"
2. 填写版本号和项目备注
3. 登录微信公众平台，提交审核
4. 审核通过后发布上线

## 安全提示

- 不要向陌生人透露个人信息
- 不要参与任何形式的"校园贷"
- 遇到可疑情况及时向辅导员或学校保卫处报告
- 合法求助渠道：96110反诈热线、学校保卫处、辅导员
